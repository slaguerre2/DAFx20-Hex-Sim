DAFx20-Hex-Sim

Sample audio files accompanying the paper "Simulating a Hexaphonic Pickup using Parallel Comb Filters for Guitar Distortion" by Sebastian Laguerre and Gary P. Scavone.

Distorted pairwise note combinations of open string notes. Each signal is normalized to −12 dB RMS post-distortion.

Click on the section of a track to quickly jump between samples.

Sample Input Description
Mono distortion Mono The mono signal is distorted by a non-linear function.
Simulated distortion Mono The same mono signal is separated into harmonic signals by the parallel comb filter bank prior to distortion.
Hexaphonic distortion Hexaphonic The individual strings of the input signal are distorted separately.
